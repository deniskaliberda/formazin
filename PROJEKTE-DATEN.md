## WICHTIG: Bilder-Organisation

Für jedes Projekt:
1. Erstelle einen Ordner: `Projekte/[projektname]/`
2. Lege dort die Bilder ab:
   - `Hero.jpg` (Hauptbild)
   - `galerie-1.jpg`, `galerie-2.jpg`, etc.
3. Schreib die Projekt-Infos in eine Textdatei im gleichen Ordner

**Beispiel:**
```
Projekte/
  vereinsheim-ahrensfelde/           ✓ FERTIG
    - Hero.jpg
    - galerie-1.jpg
    - galerie-2.jpg
    - beschreibung.txt
  
  rathaus-ahrensfelde/               TODO
    - Hero.jpg
    - galerie-1.jpg
    - beschreibung.txt
```
